# Club category icons

Drop the SVG files here, named exactly as below (the intake tiles load these):

- driver.svg    (driver)
- woods.svg     (fairway + hybrid)
- iron_set.svg  (iron set)
- wedge.svg     (wedge)
- putter.svg    (putter)

"Other" uses a built-in drawn glyph (no file needed). Any missing file falls
back to the drawn glyph automatically.
